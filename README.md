# Galería de Arte - TuPrimeraPaginaSalazar

## Descripción
Web en Django para gestionar una galería de arte.

## Cómo probar el proyecto

1. Activar entorno virtual:
source venv/bin/activate

2. Correr el servidor:
python manage.py runserver

3. Abrir en el navegador: http://127.0.0.1:8000

## Funcionalidades y orden de prueba

1. **Inicio** - http://127.0.0.1:8000
2. **Crear Artista** - Completar nombre, nacionalidad y biografía
3. **Crear Obra** - Completar título, año y técnica
4. **Crear Movimiento** - Completar nombre, período y descripción
5. **Buscar Artista** - Buscar por nombre
